MeshColorRender
===============

A simple tool to render obj mesh with per-vertex colors

<b>Platofrm:</b> MSVC

<b>Usage:</b>

Input:

path_to_obj_file path_to_color_file

e.g.: ../fig/example1.obj ../fig/color_example1_1.txt

Mouse:

Right click: rotate

Left click: pan

Middle click: zoom

Keyboard:

"s": save current view point to file

"l": load view point from file
